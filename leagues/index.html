How to:
  • Go to the registration page to set create a username and password.
  • Visit "https://itswill.org/leagues?carousel&username=your username" to see your data.
  • The username is used to view your data. The username + password are necessary to update your data. Don't lose your account info!
Updating data:
  • The only way to update the data is through an API call.
    This is intended to be set up as a Nightbot command for a twitch stream.
  • You can setup the command like this:

    !commands add !<command-name> -ul=moderator $(urlfetch https://leagues.itswill.org/api/v1/update?username=<your-username>&password=<your-password>&column=$(1)&value=$(2))

    Be careful setting up with this method! You don't want to post your password in chat. You may want to set up through the nightbot dashboard.
  • If you're setting up the command through the Nightbot dashboard, just copy

    $(urlfetch https://leagues.itswill.org/api/v1/update?username=<your-username>&password=<your-password>&column=$(1)&value=$(2))

    into the Message field. Make sure you make it moderator-only.
  • After you have set up the command, you can update your data. Call the command like:
    !<command-name> <column> <value>
Command Parameters
Find the <column> you want to edit along the top of the table. Whichever option in that column you want to set it to you will use the value from the first column.
For example, to set region 3 to Kourend, you would type in chat:
!<command-name> region3 8
value
